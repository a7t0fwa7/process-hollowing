# Process Hollowing
Process Hollowing PoC to make gameplay enhancers undetected
## Usage
1. Clone the repo based on your executables architecture
2. Turn your executable into sh3llc0de using HxD
3. Paste it into `sh3llc0de.h`
4. Compile
